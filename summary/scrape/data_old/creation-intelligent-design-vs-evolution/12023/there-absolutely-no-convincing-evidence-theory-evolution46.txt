           Perhaps but I could think of no better term. I was calling it guided selection but that doesn't make sense.
           <br/>
           <br/>
           The essence of the term is giving a namespace to the combined effect of natural selection + random mutation.
           <br/>
           <br/>
           Obviously this
           <i>
            <b>
             is
            </b>
           </i>
           the proposed mechanism of evolution but it would hardly do to use that word, especially when I am refering mostly to a test to see if the process is acting or not.
           <br/>
           <br/>
           The idea is that since evolution predicts a pattern in genetic change what can we use to identify exactly what patterns to expect; well the answer to that is that beneficial mutations are selected for so beneficial change is the meta pattern one would expect.
           <br/>
           <br/>
           How about more than a single mutation though, how about the accumulation of benefit which is as necessary to evolution as the singular selection?
           <br/>
           <br/>
           Can one tell what the pattern in multi-mutation (having intermediates) will be?
           <br/>
           <br/>
           What would you call change that is produced by evolution and change that is not (such as genetic drift)?
           <br/>
           <br/>
           Well I said since evolution is 'selecting' amongst various mutations (presented to it via individuals) the total change is guided by these selection events.
           <br/>
           <br/>
           I suppose you could say
           <i>
            determined by
           </i>
           instead of guided but I still think guided describes it better.
