           ON TYPES OF SYSTEMS:
           <br/>
           I concluded all systems in biology are either IC or HIC
           <br/>
           <br/>
           HIC = hybrid irreducibly complex, meaning as system which contains both necessary parts and unnecessary but merely helpful parts.
           <br/>
           <br/>
           HIC is a favorite escape for arguing against IC, examples of HIC are given in which the necessary component becomes necessary, it then becomes IC.
           <br/>
           <br/>
           It is then asserted that since an HIC system can be converted to an IC system and IC system is not IC.
           <br/>
           <br/>
           The blade cuts both ways though, an IC system may be converted to an HIC system and every system has an IC core which means that all novel function is IC.

           ON INTERMEDIATES:

            any

           Most important of these being when he says 'function' he does not mean any function he means the function of the said system.
           <br/>
           <br/>
           The ability to decompose a system into systems with different function does not mean the system was not IC because you in fact are not able to remove and of the parts and retain
           <b>
            the
           </b>
           function, merely
           <b>
            a
           </b>
           function.
           <br/>
           <br/>
           Further the ability to compose a system into simpler systems with the same function does not mean the system was never IC, it means it is HIC now.
           <br/>
           <br/>
           ON GUIDED MUTATION:
           <br/>
           The tool previously mentioned is a formula for identifying the concept IC is talking about.
           <br/>
           <br/>
           I do not wish to restate it's definition if you are not interested.
           <br/>
           <br/>
           I can summarize what it does, it returns a boolean value which is an absolute measure of whether a genetic change is produced by evolution or pure chance.
           <br/>
           <br/>
           true = guided = the result of natural selection + random mutation = rational evolution.
           <br/>
           <br/>
           false = unguided = luck = irrational evolutions production of novelty
           <br/>
           <br/>
           Using this function one can define IC in another way:
           <br/>
           <br/>
           An IC system is one which returns false when measuring from any previously existing genetic code to the systems minimal functional form (IC core).
           <br/>
           <br/>
           Since nearly axiomatic that all systems have an IC core the analysis seems silly to take to see if an IC core is IC but it will still return the correct value, it is much more useful to see if a proposed IC system is just IC or HIC.
           <br/>
           <br/>
